

## 오브젝트란?

- 자바에서 오브젝트는 클래스의 인스턴스 혹은 배열이다.
- 클래스는 오브젝트를 만들기 위한 설계도

## 의존관계?

- dependency
- A --> B , A가 B에 의존한다
- Client --> Supplier 
  - Client 가 기능하려면, Supplier 가 필요하다
    - 중요한 건, Supplier 가 변경되면, Client 코드가 영향을 받는다.
  - Client 가 Supplier 를 사용, 호출, 생성, 인스턴스화, 전송
- 오브젝트 사이에서도 의존관계가 있다.
- 클래스 레벨의 의존관계와, 런타임의 의존관계가 다를 수 있음
